How to write your article (draft)

Dec 31, 2018 1 min read

This post says it all:

Ready to write a manuscript but not sure how?
-find a well-written paper in the same journal (or in similar format) to where you plan to submit
-create a “reverse outline” by asking what is the purpose of each paragraph (or sentence!)
-write your manuscript using this template

— Jen Heemstra (@jenheemstra) December 2, 2018

This approach has saved me so many times when I knew what I wanted to write about, but was stuck how to write it.

I keep a folder of papers I found to be some combination of:

  1. Clear
  2. Enjoyable
  3. Smart
  4. Concise
  5. Visionary

and work from those outlines when I’m in need of inspiration.

For more good advice, follow @jenheemstra and #AcademicTwitter!
